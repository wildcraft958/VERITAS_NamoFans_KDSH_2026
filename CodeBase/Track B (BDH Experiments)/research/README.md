# Research Directory

This directory contains research materials, reference implementations, and documentation related to the BDH narrative reasoning project.

## Contents

- **Problem_Statement_-_Kharagpur_Data_Science_Hackathon_2026.pdf**: Official problem statement
- **Task.md**: Task description and requirements
- **2509.26507v1.pdf**: BDH research paper
- **bdh/**: Original BDH implementation from Pathway
- **bdh_edufork/**: Educational fork with visualizations
- **bdh-transformers/**: HuggingFace Transformers compatible implementation
- **dynamic_vocab_bdh/**: Dynamic vocabulary extension

## Reference Implementations

These implementations were used as references and merged into the main codebase:
- `bdh-transformers`: Base for unified BDH model (recurrent attention, HuggingFace integration)
- `bdh_edufork`: Visualization utilities and educational examples
- `dynamic_vocab_bdh`: Dynamic vocabulary features
